본 시스템은 "HTML5+jQuery" 서적의 다양한 예제를 직접 개발하고 테스트 할 수 있는 클라우드 기반의 통합 개발환경입니다. "튜토리얼 시작"을 누르면, 간단한 시스템 사용방법을 안내 받으실 수 있습니다.
책의 예제를 테스트하기 위해 새 프로젝트를 만들어 봅시다.
상단 좌측에 "새 프로젝트" 버튼을 클릭하여 새 프로젝트 창을 띄웁니다.
새 프로젝트 창이 열리면, 원하는 예제 프로젝트를 선택하고 "다음" 버튼을 누릅니다.
원하는 프로젝트 이름과 내용을 입력한 후, "확인" 버튼을 눌러 프로젝트를 생성합니다.
프로젝트가 성공적으로 생성되었습니다! 생성된 프로젝트 파일은 좌측 탐색창을 통해 확인 할 수 있습니다.
이번엔 해당 프로젝트를 실행시키도록 하겠습니다. 상단 "실행" 버튼을 누릅니다.
실행된 프로젝트는 새 창을 통해 확인할 수 있습니다. "실행" 버튼을 누를때마다 해당 창이 새로고침되며, 파일을 편집하고 저장할 때마다(단축키 Ctrl+S) 자동으로 화면을 갱신합니다.

지금까지 프로젝트를 생성하고 실행하는 과정을 간단하게 살펴보았습니다. 더 자세한 사용방법은 도움말이나 구름 홈페이지(http://goorm.io)를 참고하시기 바랍니다.
